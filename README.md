🌐 외부 검색 및 URL 분석 지침 (Gemini CLI 활용)

Claude의 기본 검색 도구 대신 이미 설치된 Gemini CLI를 사용하여 토큰 소모를 최소화하고 최신 정보를 가져옵니다.

검색 수행 방식: * 웹 검색, 특정 URL 내용 분석, 혹은 최신 라이브러리 정보가 필요할 때 Claude Code의 기본 Search 도구를 사용하지 마십시오.

대신 터미널에서 gemini "[질문 및 URL]" 명령어를 직접 실행하여 결과를 얻으십시오.

명령어 실행 예시:

일반 검색: gemini "SwiftUI의 최신 NavigationStack 모범 사례에 대해 검색하고 요약해줘"

URL 분석: gemini "이 페이지의 내용을 분석해서 핵심 API 사양만 알려줘: [URL]"

데이터 최적화:

Gemini CLI의 출력 결과 중 현재 작업에 꼭 필요한 정보만 추출하여 컨텍스트에 반영하십시오.

가져온 데이터가 너무 길어 컨텍스트가 비대해졌다고 판단되면 즉시 /compact 명령어를 실행하여 대화를 요약하십시오.

우선순위:

공식 문서 확인 > Gemini CLI 검색 > Claude 내부 지식 순으로 신뢰하되, 외부 데이터 유입 시 반드시 Gemini CLI를 거치도록 합니다.
